# <a name="numeric-literals"></a>Valori letterali numerici
I valori letterali numerici vengono usati quando i valori di dati numerici vengono archiviati in stringhe di caratteri. Per la conversione di dati SQL numerici in una stringa SQL_C_CHAR o la conversione di dati C numerici in una stringa SQL_CHAR o SQL_VARCHAR, viene usata la sintassi numerica letterale per specificare gli elementi archiviati nella destinazione. Per la conversione di un valore numerico archiviato come stringa di SQL_C_CHAR in dati SQL numerici o un valore numerico archiviato come stringa di SQL_CHAR in dati numerici C, questa sintassi viene usata per convalidare ci√≤ che viene archiviato nell'origine.  
  
 I valori letterali numerici devono essere conformi alla sintassi definita nella sezione relativa alla [sintassi dei valori letterali numerici](../../../odbc/reference/appendixes/numeric-literal-syntax.md) nell'Appendice C: grammatica SQL.
