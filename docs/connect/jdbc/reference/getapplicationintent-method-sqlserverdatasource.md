# <a name="getapplicationintent-method-sqlserverdatasource"></a>Metodo getApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce il valore della proprietà di connessione **applicationIntent**.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
public java.lang.String getApplicationIntent();  
```  
  
## <a name="return-value"></a>Valore restituito  
 Restituisce il valore della proprietà di connessione **applicationIntent**.  
  
## <a name="remarks"></a>Osservazioni  
 Per altre informazioni sulla proprietà di connessione **applicationIntent**, vedere [Impostazione delle proprietà di connessione](../../../connect/jdbc/setting-the-connection-properties.md).
