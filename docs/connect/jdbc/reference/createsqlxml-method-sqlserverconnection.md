# <a name="createsqlxml-method-sqlserverconnection"></a>Metodo createSQLXML (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crea un oggetto **SQLXML** senza dati.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.sql.SQLXML createSQLXML()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Oggetto SQLXML.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo createSQLXML viene specificato dal metodo createSQLXML nell'interfaccia java.sql.Connection.
