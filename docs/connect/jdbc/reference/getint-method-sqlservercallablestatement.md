# <a name="getint-method-sqlservercallablestatement"></a>Metodo getInt (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come oggetto **int** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[getInt (int)](../../../connect/jdbc/reference/getint-method-int.md)|Recupera il valore del parametro designato come **int** nel linguaggio di programmazione Java in base all'indice del parametro.|  
|[getInt (java.lang.String)](../../../connect/jdbc/reference/getint-method-java-lang-string.md)|Recupera il valore del parametro designato come **int** nel linguaggio di programmazione Java in base al nome del parametro.|
