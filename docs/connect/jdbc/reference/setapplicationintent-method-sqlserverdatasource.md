# <a name="setapplicationintent-method-sqlserverdatasource"></a>Metodo setApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il valore della proprietà di connessione **applicationIntent**.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>Parametri  
 *applicationIntent*  
  
 Nuovo valore della proprietà di connessione **applicationIntent**.  
  
## <a name="remarks"></a>Osservazioni  
 Per altre informazioni sulla proprietà di connessione **applicationIntent**, vedere [Impostazione delle proprietà di connessione](../../../connect/jdbc/setting-the-connection-properties.md).
