# <a name="position-method-sqlserverclob"></a>Metodo position (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce la posizione del carattere dell'oggetto CLOB specificato o della sottostringa in tale oggetto in base alla posizione di inizio specificata.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[position (java.sql.Clob, long)](../../../connect/jdbc/reference/position-method-java-sql-clob-long.md)|Restituisce la posizione del carattere dell'oggetto CLOB specificato in tale oggetto in base alla posizione di inizio specificata.|  
|[position (java.lang.String, long)](../../../connect/jdbc/reference/position-method-java-lang-string-long.md)|Restituisce la posizione del carattere della sottostringa specificata nell'oggetto CLOB in base alla posizione di inizio specificata.|
