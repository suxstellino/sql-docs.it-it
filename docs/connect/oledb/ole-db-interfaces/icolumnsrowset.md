# <a name="icolumnsrowset"></a>IColumnsRowset
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  In OLE DB Driver per SQL Server viene aggiunta la colonna DBCOLUMN_BASETABLEINSTANCE a IColumnsRowset::GetColumnRowset. Questa colonna restituisce DBTYPE_I2 ed è riservata per l'uso da parte di Microsoft. Le informazioni in questa colonna sono soggette a modifiche nelle versioni future.
