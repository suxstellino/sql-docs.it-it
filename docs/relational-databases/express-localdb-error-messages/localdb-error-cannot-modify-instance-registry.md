# <a name="localdb_error_cannot_modify_instance_registry"></a>LOCALDB_ERROR_CANNOT_MODIFY_INSTANCE_REGISTRY
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
    
## <a name="details"></a>Dettagli  
  
| Attributo | Valore |
| --------- | ----- |
|Nome prodotto|SQL Server|  
|ID evento|265|  
|Origine evento|Runtime database locale di SQL Server 12.0|  
|Componente|API di Runtime database locale|  
|Testo del messaggio|Errore imprevisto durante il tentativo di modifica della configurazione del Registro di sistema per l'istanza del database locale. Vedere il registro eventi applicazioni di Windows per informazioni sull'errore.|  
  
## <a name="explanation"></a>Spiegazione  
 Impossibile modificare il Registro di sistema di un'istanza.
