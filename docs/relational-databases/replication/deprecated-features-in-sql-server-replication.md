# <a name="deprecated-features-in-sql-server-replication"></a>Funzionalità deprecate nella replica di SQL Server
[!INCLUDE [SQL Server SQL MI](../../includes/applies-to-version/sql-asdbmi.md)]
  In questo argomento vengono descritte le funzionalità deprecate della replica ancora disponibili in [!INCLUDE[ssnoversion](../../includes/ssnoversion-md.md)]. Tali funzionalità verranno rimosse a partire da una delle prossime versioni di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. È consigliabile non usare le funzionalità deprecate nelle nuove applicazioni.  
  
## <a name="items-deprecated-in-sssql16-md"></a>Elementi deprecati in [!INCLUDE[sssql16-md](../../includes/sssql16-md.md)]  
  
|Funzionalità|Descrizione|  
|-------------|-----------------|  
|[!INCLUDE[ssKatmai](../../includes/sskatmai-md.md)]|La replica è supportata se ogni endpoint di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] rientra nelle due versioni principali della versione corrente di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Pertanto, [!INCLUDE[sssql16-md](../../includes/sssql16-md.md)] non supporta la replica in o da [!INCLUDE[ssKatmai](../../includes/sskatmai-md.md)] o [!INCLUDE[ssKilimanjaro](../../includes/sskilimanjaro-md.md)].|  
|[!INCLUDE[ssEW](../../includes/ssew-md.md)]|La replica è supportata se ogni endpoint di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] rientra nelle due versioni principali della versione corrente di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Pertanto, [!INCLUDE[sssql16-md](../../includes/sssql16-md.md)] non supporta la replica in o da [!INCLUDE[ssEW](../../includes/ssew-md.md)].|
